# 概要
Pythonのコンソールアプリで、JSONファイルを使った  
**簡易的なユーザー登録・一覧・削除ツール**です。

---

## 機能
✅ 氏名を入力してユーザーを登録  
✅ 登録済ユーザーの一覧表示  
✅ IDを指定してユーザーを削除  
✅ データは `data.json` に自動保存  

---

## 構成

├─ main.py # メインアプリ（コンソール版）
├─ data.json # データ保存用（自動生成）
└─ README.md

---

## 実行方法

### 必要環境
- Python 3.x

### 実行コマンド
```bash
python main.py
