# 概要
Tkinter（Python標準GUIライブラリ）とJSONファイルを使った、  
**簡易的なユーザー登録・一覧・削除ツール**のサンプルです。

---

## 機能

✅ 氏名を入力して登録  
✅ 登録済ユーザーの一覧表示  
✅ 選択したユーザーを削除  
✅ データは `data.json` に自動保存  

---

## 構成

 ┣ tkinter_json_crud.py
 ┣ data.json
 ┗ README.md

---

## 実行方法

### 必要環境
- Python 3.8 以上  
（Tkinter は標準で含まれています）

### 実行コマンド
```bash
python tkinter_json_crud.py
